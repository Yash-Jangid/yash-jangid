## 🧠 How I build (Developer OS)
- Prefer **event-driven** designs over “one giant cron that prays”
- Reliability basics: **idempotency, retries, backoff, DLQs, timeouts**
- Keep systems boring in the best way: **clear boundaries, clean APIs**
- Performance is a feature: **cache smart, batch often, index always**
- Ship fast, then harden: **observe → fix → automate**
